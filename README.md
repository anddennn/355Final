# Reddit Sentiment Visualizations - IAT 355 Final Assignment

Vega-Lite visualizations analyzing Reddit post sentiment from 2015-2025.

## Setup

1. Start a local server:
   ```bash
   python3 -m http.server 8000
   ```

2. Open `http://localhost:8000` in your browser.

## Files

- `index.html` - Main page
- `main.js` - Visualization specs
- `Reddit Posts 2015-2025.csv` - Dataset

## Visualizations

1. Subreddit popularity over time (line chart)
2. Average sentiment by subreddit (bar chart)
3. Sentiment vs. score relationship (scatterplot with interactive filtering)
